# QQHongBao
This is a RedPacket Plugin


QQ抢红包思路：
  三种情况：
    1、界面状态改变
    2、界面内容改变
    3、通知栏处理
    
    界面状态改变和界面内容改变又分两种情况：
      1、当前界面是消息内容界面
          判断是否有红包：
            无：不处理
            有：记录红包true
                模拟点击
                记录红包false
                如果红包为false则返回
                
      2、当前界面是消息列表界面
          判断是否有未读消息：
            无：不处理
            有：进入未读消息
                判断是否有红包：记录红包true/false
                  无：返回（false）
                  有：(true)
                      模拟点击
                      记录红包false
                      如果红包为false则返回
                   继续判断。。。
